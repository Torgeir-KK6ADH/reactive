Python SDK Notes

The Python Rapid SDK is designed to help in creating Python applications to be executed on the Sirit UFH readers.

Executing a Python application on the Reader

  1. Open a browser
  2. Go to URL http://reader_ip_address
  3. Select User Application Manager under Advanced Functions
  4. Under Application Transfer, enter in the location of the rapid.py file. By default this file will be in the RAPID SDK\python directory of the RST install directory. Transfer the file to reader by pressing the Transfer File button. This step will only need to be done the first time or when the rapid.py has changed. This file will now be listed in the Application Name combo box and the Python Applications combo box
  5. Under Application Transfer, enter in the location of the application class file. Transfer the file to reader by pressing the Transfer File button. The application will now be listed in the Application Name combo box and the Python Applications combo box
  6. Under Python Applications combo box, select the application.
  7. Press the Go button. A Message box will be displayed showing the process ID for the application. When the screen refreshes, the process ID will be listed under Running User Applications.

Python Developer Requirements

  • The first statement of a python file needs to be #!/usr/bin/python -u
  • To load the rapid SDK use import rapid command
